Objetivo: O objetivo deste trabalho é projetar,desenvolver e avaliar uma aplicação web de comércio eletrônico especializada na venda de produtos geeks. A ideia principal é juntar num lugar só uma tonelada de produtos de várias franquias geek para que os fãs possam encontrar tudo o que gostam em um lugar só na internet.

Justificativa: Este site foi desenvolvido com o propósito de proporcionar aos entusiastas da cultura geek a conveniência de localizar e adquirir todos os seus produtos e acessórios preferidos em uma únicaplataforma centralizada.

Escopo do Sistema: Será uma loja online de venda de produtos. Para poder realizar uma compra o cliente deverá estar cadastrado em nosso site onde coletamos as informações necessárias para mandar o produto ao cliente, os dados coletados são: nome, sobrenome, Email, cpf e telefone para identificar e diferenciar cada cliente e colocar seus produtos em carrinhos antes do pagamento ser feito, depois que o cliente tiver escolhido todos os seus produtos, ele poderá registrar o cartão de crédito para efetuar o pagamento, e enviará para o endereço que o cliente definiu e que é armazenado no banco de dados.
